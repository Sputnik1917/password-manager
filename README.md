
Учебный проект на **Spring Boot**, демонстрирующий работу с базой данных PostgreSQL и реализацию функции **смены пароля** для существующего пользователя.

---

## 🚀 Возможности
- Добавление и хранение клиентов в базе данных  
- Смена пароля по `clientId`  
- Работа через REST API  
- Подключение к PostgreSQL через JPA/Hibernate
